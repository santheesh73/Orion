# Folder Structure

```text
app/                  Next.js routes and route-level states
components/common/    Shared product components
components/layout/    Shell, navigation, panels, containers
components/pwa/       PWA status and performance dashboard
components/ui/        Reusable UI primitives
features/ai/          Browser local AI client
features/chat/        Chat domain components and types
hooks/                Reusable React hooks
lib/                  Constants, utilities, theme, database connection
providers/            App and theme providers
repositories/         IndexedDB repository layer
services/             AI, document, storage, settings, PWA, performance services
store/                Ephemeral UI state
workers/              Browser workers for LLM and document processing
public/               Manifest, service worker, icons, screenshots, docs assets
docs/                 Project documentation
```
